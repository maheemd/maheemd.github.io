Why this website exists

Knowledge becomes more valuable when it is organized, preserved and made useful to other people.

My path has crossed engineering, entrepreneurship and energy-economics research. Along the way, I have learned through formal study, experiments, client projects, difficult mistakes and long periods of independent work.

This website is a permanent home for that journey. It will collect my research, publications, project documentation, teaching materials and practical notes. Some resources will be academic; others will return to the foundations of robotics, electronics, Arduino and IoT.

The goal is not only to present a professional profile. It is also to make sure that useful knowledge does not disappear when a project ends, a social-media account changes or a paid website is discontinued.

Build carefully, document clearly and leave something that another person can continue.

This GitHub Pages website is intentionally lightweight and open. Its source code can be maintained without a database, subscription or specialized website administrator.
